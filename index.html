I'm A Hacker

My raspberry pi runs my sprinkler system so I can schedule it via Google Calendar. It also senses the position of my garage door and allows me to trigger it from my cell phone (can you say 'security risk?'). My other pi used to be a Tor node but is now my computer vision experimentation platform. The point I'd like to make here is that I'm excited about physical computing and I see UI/UX as much, much more than a web page in a browser. I'd kill to work on your projects.
